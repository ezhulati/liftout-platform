# User Journeys: End-to-End Platform Flows

Complete user journey maps for both Liftout user types, from discovery to advocacy.

---

## Journey 1: Team User (Alex Chen)

### Phase 1: Awareness
**Timeline**: Days 1-7

```
┌─────────────────────────────────────────────────────────────────┐
│  TRIGGER: Team member forwards article about group laterals     │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  RESEARCH: Googles "team hiring" "group lateral moves"          │
│  Finds: HBR article, Groysberg research, Liftout                │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  FIRST VISIT: Lands on liftout.com                              │
│  Reads: "What is a Liftout?" page                               │
│  Emotion: Curiosity → "This is a real thing?"                   │
└─────────────────────────────────────────────────────────────────┘
```

**Key Pages Visited**:
- Homepage → Hero message: "The job board for teams"
- `/what-is-a-liftout` → Definition, examples, research
- `/for-teams` → Team-specific value proposition

**Conversion Blockers**:
- "Is this legit?" → Trust signals (HBR, statistics)
- "Will my employer find out?" → Confidentiality messaging
- "Are there actually opportunities?" → Browse without signing up

## Journey Comparison

| Phase | Team User (Alex) | Company User (Sarah) |
|-------|------------------|---------------------|
| **Trigger** | Career stagnation + team loyalty conflict | CEO deadline + hiring frustration |
| **Discovery** | Google → HBR → Liftout | Peer referral → Research → Liftout |
| **Consideration** | 2 weeks (team consensus needed) | 1 week (internal approval needed) |
| **Signup** | Solo then invites team | Company profile + verification |
| **Active Use** | Browse opportunities, manage visibility | Post opportunities, search teams |
| **Conversion** | 8-12 weeks (interview + negotiation) | 4-8 weeks (evaluate + hire) |
| **Success Metric** | Team moves together + better comp | Team productive in <30 days |
| **Advocacy** | Word of mouth to other team leads | Case study + peer referrals |

## Critical Moments (Make or Break)

### For Team Users
1. **Team buy-in conversation** - If one member says no, the journey ends
2. **First inquiry from company** - Validates the platform works
3. **Joint interview** - Chemistry must be visible
4. **Package negotiation** - Must work for all team members

### For Company Users
1. **Internal approval** - CEO/CFO/Legal must align
2. **First team response** - Validates quality of supply
3. **Reference checks** - Team must be as good as profile claims
4. **Offer acceptance** - All team members must sign
